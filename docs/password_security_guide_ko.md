# VisionChain 비밀번호 및 보안 업데이트 안내

## 변경 사항

### 로그인 비밀번호와 지갑 비밀번호 분리
계정에 **두 개의 독립된 비밀번호**가 적용됩니다:

- **로그인 비밀번호** -- 이메일로 VisionChain 계정에 로그인할 때 사용
- **지갑 비밀번호** -- 지갑 암호화/복호화 및 클라우드 백업, 복구에 사용

**설정 > 비밀번호**에서 각각 독립적으로 변경할 수 있습니다.

### 클라우드 백업은 지갑 비밀번호를 사용합니다
**Sync to Cloud** 기능은 **지갑 비밀번호**(로그인 비밀번호가 아님)로 지갑을 암호화합니다:

- 지갑 백업 시 **지갑 비밀번호**를 입력합니다.
- 새 기기에서 지갑 복구 시 **지갑 비밀번호**가 필요합니다.
- 지갑 비밀번호를 변경하면 클라우드 백업이 자동으로 새 비밀번호로 업데이트됩니다.

### 비밀번호 변경 시 2FA(Google OTP) 필수
계정 보호를 위해 비밀번호 변경 전 반드시 **Google Authenticator(2FA) 인증이 필요**합니다. 2FA를 아직 설정하지 않았다면 보안 탭에서 먼저 설정해야 합니다.

### 지갑 복구 시 2FA 필수
새 기기에서 클라우드 지갑을 복구할 때 지갑 비밀번호 외에 **2FA 인증이 추가로 필요**합니다. 비밀번호가 유출되더라도 인증 코드 없이는 지갑에 접근할 수 없습니다.

---

## 중요 안내

> **모든 사용자는 반드시 2단계 인증(2FA)을 활성화할 것을 강력히 권장합니다.**
>
> 2FA를 설정하지 않은 상태에서 비밀번호를 분실할 경우, 자금 손실 또는 지갑 접근 불가에 대한 **모든 책임은 사용자 본인에게 있습니다.**
>
> VisionChain은 적절한 인증 없이 비밀번호, 지갑 또는 자금을 **복구할 수 없습니다.** 지갑 암호화에 대한 비밀번호 초기화 기능은 존재하지 않으며, 지갑 비밀번호와 2FA만이 자산에 접근하는 유일한 방법입니다.

## 빠른 참조

| 작업 | 사용하는 비밀번호 | 2FA 필요? |
|------|-----------------|----------|
| 계정 로그인 | 로그인 비밀번호 | 아니오 |
| 로그인 비밀번호 변경 | 로그인 비밀번호 + OTP | **필수** |
| 지갑 비밀번호 변경 | 지갑 비밀번호 + OTP | **필수** |
| 클라우드 백업 | 지갑 비밀번호 | 아니오 |
| 클라우드 지갑 복구 | 지갑 비밀번호 + OTP | **필수** |
| 송금 | 지갑 비밀번호 | 아니오 |

---

## 시작하기

1. **설정 > 보안**에서 **Google Authenticator(2FA)**를 활성화하세요.
2. **백업 코드**를 안전한 곳에 보관하세요.
3. **설정 > 비밀번호**에서 로그인 비밀번호와 지갑 비밀번호를 각각 관리하세요.
4. **설정 > 보안 > Sync to Cloud**에서 지갑 비밀번호로 백업하세요.
