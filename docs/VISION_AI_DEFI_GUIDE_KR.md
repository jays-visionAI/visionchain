# 🚀 비전 AI: 지능형 DeFi 포트폴리오 전략가 가이드

탈중앙화 금융(DeFi)의 미래에 오신 것을 환영합니다. 비전 AI는 이제 단순한 챗봇을 넘어, 당신의 개인용 온체인 투자 전략가로 진화했습니다.

이 가이드는 현재 비전 AI 터미널에서 활성화된 새로운 **DeFi Pro** 기능을 소개합니다.

---

## 💎 핵심 기능

### 1. 알파 발견 (DeFi 수익률 스캐너)
비전 AI는 이더리움, 솔라나, 폴리곤 등 멀티체인에 걸친 수천 개의 유동성 풀, 렌딩 마켓, 스테이킹 프로토콜을 실시간으로 스캔하여 당신의 자산에 대한 최고 수익률(APY)을 찾아냅니다.
*   **질문 예시**: "지금 내 USDC로 가장 높은 수익을 얻을 수 있는 곳은 어디야?"
*   **질문 예시**: "TVL 1억 달러 이상의 ETH 스테이킹 풀을 찾아줘."

### 2. 지능형 리스크 평가
높은 수익에는 대개 높은 위험이 따릅니다. 비전 AI는 투자처를 제안하기 전, 해당 프로토콜의 보안 상태와 안정성을 평가하는 휴리스틱 감사(Heuristic Audit)를 수행합니다.
*   **질문 예시**: "Lido 프로토콜은 사용하기에 안전해?"
*   **질문 예시**: "폴리곤 체인 Aave의 위험 수준을 분석해줘."

### 3. 과거 성과 및 시장 비교 분석
오늘의 기회를 과거 데이터와 비교하여 더 현명한 결정을 내릴 수 있도록 돕습니다.
*   **질문 예시**: "지난 1년간의 비트코인 가격 상승률과 일반적인 DeFi 스테이블코인 수익률을 비교해서 보여줘."

## 🛠 작동 원리

비전 AI는 **실시간 데이터 주입(Real-Time Data Injection)** 및 **자율 도구 실행(Tool Execution)** 기술을 사용합니다.
1.  **의도 분석(Intent Parsing)**: 단순한 키워드가 아닌, 금융 문맥 전체를 이해합니다.
2.  **도구 오케스트레이션**: CoinGecko, DeFi Llama 등 전문 서비스의 API를 호출하여 실시간 데이터를 확보합니다.
3.  **리스크 필터링**: TVL 규모와 변동성 지표를 평가하여 기회의 안정성 점수를 산출합니다.
4.  **인사이트 요약**: 복잡한 데이터를 누구나 이해하기 쉬운 명확하고 실행 가능한 조언으로 요약합니다.

---

## 🛡 보안 원칙 (Security First)

*   **비수탁형(Non-Custodial)**: 비전 AI는 최적의 거래를 '제안'할 뿐, 모든 실행은 유저의 직접 승인(서명)을 거칩니다. 우리는 유저의 개인키에 절대 접근하지 않습니다.
*   **투명성**: 모든 조언은 데이터에 기반한 논리적 근거를 포함합니다.
*   **위험 경고**: TVL이 낮거나 검증되지 않은 신규 프로토콜에 대해서는 시스템이 명확하게 경고 메시지를 표시합니다.

## 🕒 Time Lock Agent 테스트 가이드 (New)

Vision AI의 타임락(예약 송금) 기능을 빠르고 면밀하게 테스트하는 방법입니다.

### 1. 예약 거래 생성
AI 채팅창에 다음과 같이 입력하세요:
> "1분 뒤에 [받는사람]에게 10 VCN 보내줘"
> "오후 7시에 [받는사람]에게 50 VCN 예약해줘"

AI가 요청을 이해하고 예약 카드를 생성하여 우측 대기열에 추가합니다.

### 2. 대기열(Queue) 확인
화면 우측 상단의 **[Agent Queue]** 버튼을 클릭하거나, 자동으로 열리는 사이드바에서 예약된 내역을 확인합니다.
- 상태: `WAITING` (파란색)
- 남은 시간: `In 1m` 등 표시

### 3. 테스트 실행 방법

#### A. 자동 실행 테스트 (Standard)
1. 예약된 시간이 될 때까지 기다립니다.
2. 시간이 되면 상태가 즉시 `EXECUTING`으로 변경되고, 트랜잭션이 전송됩니다.
3. 완료되면 `SENT` 상태로 바뀌고 TX Hash가 생성됩니다.

#### B. 즉시 강제 실행 (Debug Mode) ⚡️
기다리지 않고 로직이 정상 동작하는지 바로 확인하고 싶을 때 사용하세요.
1. Queue 패널의 예약 카드에서 **초록색 재생(▶) 버튼**을 클릭합니다.
2. 시간이 무시되고 즉시 트랜잭션이 실행됩니다.
3. 실행 후 상태가 `SENT`로 변하고 잔액이 차감되는지 확인하세요.

### ❓ 자주 묻는 질문 (FAQ)

**Q. Time Lock 예약 시 돈이 미리 빠져나가나요? (에스크로)**
A. 아니요, 현재 버전은 **Non-Escrow (비수탁형)** 방식입니다.
- 예약 시점에는 자금이 지갑에 그대로 남아있으며, 아무런 가스비도 들지 않습니다.
- **취소 기능**은 미래에 실행될 '자동 이체 명령'을 삭제하는 것이므로, 즉시 반영되며 수수료가 없습니다.
- 지정된 시간이 되면 Agent가 자동으로 트랜잭션을 생성하여 전송을 실행합니다. 이때 잔액이 부족하면 전송은 실패합니다.
