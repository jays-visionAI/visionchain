## 1. Executive Summary

### 1.1 비전
VisionDEX는 Vision Chain 생태계의 핵심 인프라로서, CEX 수준의 사용자 경험을 제공하는 탈중앙화 거래소입니다. TradingView 차트, 실시간 호가창, 원클릭 거래 기능을 통해 초보자부터 전문 트레이더까지 모든 사용자가 편리하게 사용할 수 있습니다.

### 1.2 핵심 차별화
| 요소 | 기존 DEX (Uniswap 등) | VisionDEX |
|------|----------------------|-----------|
| 거래 방식 | AMM (풀 기반) | Hybrid (호가창 + AMM) |
| 차트 | 외부 링크 | TradingView 내장 |
| 주문 유형 | 시장가만 | 시장가/지정가/손절/익절 |
| 유동성 | 분산됨 | 통합 (Aggregated) |
| 실행 속도 | 블록 단위 | Sub-second |
| 가스비 | 사용자 부담 | Paymaster 지원 |
| dApp 상장 | 없음 | Launchpad + 상장 |

#### 4.1.2 호가창 (Order Book)
```typescript
interface OrderBookLevel {
    price: string;      // 가격
    amount: string;     // 수량 (VCN)
    total: string;      // 누적 (USDT)
    percentage: number; // 깊이 시각화용 %
}

interface OrderBook {
    asks: OrderBookLevel[];  // 매도 주문 (가격 오름차순)
    bids: OrderBookLevel[];  // 매수 주문 (가격 내림차순)
    lastPrice: string;
    spread: string;
    spreadPercent: string;
}

// WebSocket 실시간 업데이트
const orderBookWS = new WebSocket('wss://dex.visionchain.co/ws/orderbook/VCN-USDT');

orderBookWS.onmessage = (event) => {
    const data = JSON.parse(event.data);
    updateOrderBook(data);
};
```

#### 4.1.3 주문 유형
| 주문 유형 | 설명 | 파라미터 |
|-----------|------|----------|
| **Market** | 현재가 즉시 체결 | amount |
| **Limit** | 지정가 주문 | price, amount |
| **Stop-Limit** | 트리거 가격 도달 시 지정가 | triggerPrice, price, amount |
| **Take-Profit** | 이익 실현 | triggerPrice, amount |
| **Stop-Loss** | 손실 제한 | triggerPrice, amount |

## 6. API 설계

### 6.1 REST API Endpoints
```
Base URL: https://api.dex.visionchain.co/v1

# Market Data (Public)
GET  /markets                    # 모든 마켓 목록
GET  /markets/{pair}/ticker      # 24h 통계
GET  /markets/{pair}/orderbook   # 호가창 스냅샷
GET  /markets/{pair}/trades      # 최근 체결
GET  /markets/{pair}/klines      # OHLCV 캔들

# Trading (Authenticated)
POST /orders                     # 주문 생성
GET  /orders                     # 미체결 주문
DELETE /orders/{orderId}         # 주문 취소
DELETE /orders                   # 전체 취소

# Account (Authenticated)
GET  /account/balances           # 잔액 조회
GET  /account/trades             # 거래 내역
GET  /account/orders/history     # 주문 이력
```

### 6.2 WebSocket Streams
```javascript
// 연결
const ws = new WebSocket('wss://stream.dex.visionchain.co');

// 구독
ws.send(JSON.stringify({
    method: 'SUBSCRIBE',
    params: [
        'vcn-usdt@ticker',      // 실시간 가격
        'vcn-usdt@depth20',     // 호가창 (상위 20)
        'vcn-usdt@trade',       // 체결
        'vcn-usdt@kline_1m',    // 1분봉
    ]
}));

// 메시지 수신
ws.onmessage = (event) => {
    const data = JSON.parse(event.data);
    switch(data.stream) {
        case 'vcn-usdt@ticker': updateTicker(data); break;
        case 'vcn-usdt@depth20': updateOrderBook(data); break;
        case 'vcn-usdt@trade': updateTrades(data); break;
    }
};
```
