# Vision Chain 2단계 인증(2FA) 설정 가이드

## 소개

Vision Chain 지갑은 Google Authenticator를 사용한 2단계 인증(2FA)을 지원합니다. 2FA를 활성화하면 클라우드에서 지갑을 복원할 때 추가 보안 코드가 필요하므로, 휴대폰 분실이나 해킹 시에도 자산을 안전하게 보호할 수 있습니다.

---

## 1. Google Authenticator 앱 설치

### iOS (iPhone/iPad)
1. App Store를 엽니다
2. "Google Authenticator" 검색
3. **설치** 버튼 탭

### Android
1. Google Play Store를 엽니다
2. "Google Authenticator" 검색
3. **설치** 버튼 탭

---

## 2. 2FA 활성화하기

### 단계 1: 설정 메뉴 접근
1. Vision Chain 지갑 앱을 엽니다
2. **설정** 메뉴로 이동합니다
3. **보안 설정** > **2단계 인증** 선택

### 단계 2: QR 코드 스캔
1. "2FA 활성화" 버튼을 탭합니다
2. 화면에 나타난 **QR 코드**를 확인합니다
3. Google Authenticator 앱을 열고 **+** 버튼을 탭합니다
4. **QR 코드 스캔**을 선택합니다
5. Vision Chain 앱에 표시된 QR 코드를 스캔합니다

> **QR 코드를 스캔할 수 없나요?**
> 화면에 표시된 수동 입력 키를 Google Authenticator에서 **수동 입력** 옵션으로 입력하세요.

### 단계 3: 인증 코드 확인
1. Google Authenticator에서 **Vision Chain** 항목을 찾습니다
2. 표시된 **6자리 코드**를 확인합니다
3. Vision Chain 앱에 코드를 입력합니다
4. **활성화** 버튼을 탭합니다

### 단계 4: 백업 코드 저장 (중요!)
1. 활성화 성공 시 **8개의 백업 코드**가 표시됩니다
2. 이 코드들을 **안전한 곳에 저장**하세요:
   - 종이에 적어서 보관
   - 암호화된 노트에 저장
   - 비밀번호 관리자에 저장
3. 백업 코드는 이메일로도 전송됩니다

> **주의**: 백업 코드는 Google Authenticator에 접근할 수 없을 때 사용됩니다.
> 각 백업 코드는 **한 번만** 사용할 수 있습니다.

---

## 3. 클라우드에서 지갑 복원하기 (2FA 사용)

### 2FA가 활성화된 경우
1. **클라우드에서 복원** 선택
2. 지갑 비밀번호 입력
3. **Google Authenticator 코드 입력** 화면이 나타납니다
4. Google Authenticator 앱을 열고 Vision Chain의 6자리 코드 확인
5. 코드 입력 후 **확인** 탭
6. 지갑 복원 완료!

### 코드를 입력할 수 없는 경우
1. **백업 코드 사용** 옵션을 선택합니다
2. 저장해둔 8자리 백업 코드 중 하나를 입력합니다
3. 지갑 복원 완료 (사용된 백업 코드는 재사용 불가)

---

## 4. 2FA 비활성화하기

> **경고**: 2FA를 비활성화하면 보안 수준이 낮아집니다.

1. **설정** > **보안 설정** > **2단계 인증**으로 이동
2. **2FA 비활성화** 버튼 탭
3. Google Authenticator의 현재 6자리 코드 입력
4. 비활성화 완료

---

## 5. 자주 묻는 질문 (FAQ)

### Q: 휴대폰을 분실했어요. 어떻게 하나요?
**A**: 저장해둔 백업 코드를 사용하여 지갑을 복원하세요. 복원 후 새 휴대폰에서 2FA를 다시 설정하세요.

### Q: Google Authenticator를 삭제했어요.
**A**: 백업 코드로 로그인한 후, 2FA를 비활성화하고 다시 설정하세요.

### Q: 백업 코드도 잃어버렸어요.
**A**: 고객 지원팀에 연락하세요. 본인 확인 절차를 통해 계정 복구를 도와드립니다.

### Q: 코드가 맞는데 "잘못된 코드"라고 나와요.
**A**: 
- 휴대폰 시간이 정확한지 확인하세요 (자동 시간 설정 권장)
- Google Authenticator 앱에서 시간 동기화를 실행하세요:
  - 설정 > 코드의 시간 수정 > 지금 동기화

### Q: 2FA 코드는 얼마나 자주 바뀌나요?
**A**: 30초마다 새로운 코드가 생성됩니다.

### Q: 여러 기기에서 같은 2FA를 사용할 수 있나요?
**A**: 네! 설정 시 QR 코드를 여러 기기에서 스캔하면 동일한 코드가 생성됩니다.

## 6. 보안 권장 사항

1. **백업 코드 안전 보관**: 온라인에 저장하지 마세요
2. **화면 캡처 주의**: QR 코드나 백업 코드는 캡처하지 마세요
3. **주기적 확인**: 백업 코드 잔여 개수를 주기적으로 확인하세요
4. **기기 분실 시**: 즉시 다른 기기에서 2FA를 재설정하세요
5. **의심 활동 시**: 새 기기 로그인 알림을 받으면 즉시 확인하세요
