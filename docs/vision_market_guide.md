# Vision Market 사용 가이드 -- 프리미엄 디지털 콘텐츠 마켓플레이스

## Vision Market이란?

Vision Market은 Vision Chain 생태계의 탈중앙화 디지털 콘텐츠 마켓플레이스입니다. 유저들이 자신의 데이터셋, 이미지, 영상, 문서 등을 VCN 토큰으로 거래할 수 있는 글로벌 플랫폼입니다.

---

## 주요 특징

- **탈중앙화 스토리지**: 모든 파일은 Vision Node 네트워크에 256KB 청크로 분산 저장됩니다
- **클라이언트 사이드 암호화**: AES-256-GCM으로 브라우저에서 직접 암호화하여 업로드 가능
- **VCN 토큰 결제**: 가격은 판매자가 자유롭게 설정하며, VCN 토큰으로 거래됩니다
- **영상 스트리밍**: 분산 저장된 영상 파일도 프로그레시브 버퍼링을 통해 재생 가능
- **Merkle Tree 검증**: 데이터 무결성 보장

---

## 1단계: Vision Disk 구독하기

Vision Market에서 콘텐츠를 판매하려면 먼저 **Vision Disk** 스토리지 구독이 필요합니다.

1. 지갑 좌측 메뉴에서 **Disk** 탭을 선택합니다
2. 요금제를 선택합니다 (기본 10GB -- 0.50 VCN/GB)
3. **Subscribe** 버튼을 클릭하여 VCN 토큰으로 결제합니다
4. EIP-2612 가스리스 서명으로 간편하게 결제됩니다

---

## 2단계: 파일 업로드하기

### 업로드 방법
- **Upload 버튼** 클릭 또는 **드래그앤드롭**으로 파일을 업로드합니다
- 파일당 최대 500MB까지 지원합니다

### 스토리지 옵션

| 옵션 | 설명 |
|------|------|
| **VNet (Vision Network)** | 파일을 256KB 청크로 분할하여 전 세계 비전 노드에 분산 저장합니다. 3개 이상의 노드에 복제하여 최대 내구성을 보장합니다 |
| **Private (E2E 암호화)** | 브라우저에서 직접 AES-256-GCM으로 암호화한 후 업로드합니다. 비전체인조차 내용을 볼 수 없습니다 |

> **참고**: 암호화된 파일도 썸네일은 원본 기반으로 생성되어 파일 목록에서 미리보기가 가능합니다.

## 3단계: Market에 게시하기 (판매 등록)

1. 업로드된 파일을 **우클릭** (또는 모바일에서 길게 누르기) 합니다
2. 컨텍스트 메뉴에서 **"Publish to Market"** 을 선택합니다
3. **VCN 가격**을 설정합니다 (예: 5 VCN)
4. 수수료 안내를 확인합니다:
   - 판매가의 **70%** -> 판매자 수령
   - 판매가의 **30%** -> Vision Chain 수수료
5. **"Publish Now"** 버튼을 클릭합니다
6. 게시 즉시 **모든 유저의 Market 탭**에 노출됩니다

### 게시 취소
- 파일 우클릭 -> **"Unpublish"** 를 선택하면 Market에서 즉시 제거됩니다

---

## 4단계: 콘텐츠 구매하기

1. 좌측 메뉴에서 **Market** 탭을 선택합니다
2. 원하는 콘텐츠를 검색하거나 브라우징합니다
3. **"Buy Now"** 버튼을 클릭합니다
4. VCN 잔액에서 자동 차감됩니다
5. 구매 완료 후 **"Download"** 버튼으로 파일을 다운로드합니다

> 이미 구매한 콘텐츠는 **"Purchased"** 태그가 표시되며, 언제든 다시 다운로드할 수 있습니다.

---

## 5단계: 영상 콘텐츠 재생

Vision Market에 등록된 영상 파일은 **프로그레시브 스트리밍**으로 재생됩니다.

### 영상 파일 크기 권장 사양

| 파일 크기 | 버퍼 시간 | 사용자 경험 |
|-----------|----------|------------|
| 50MB 이하 | ~10초 | 최적 -- 거의 즉시 재생 |
| 50~100MB | ~20초 | 양호 -- 짧은 대기 후 재생 |
| 100~200MB | ~40초 | 허용 -- 대용량 안내 표시 |
| 200MB 이상 | 1분+ | 비권장 -- 다운로드 후 재생 권장 |

### 재생 과정
1. 파일 클릭 시 **비디오 플레이어 UI**가 표시됩니다
2. **Buffering** 단계: 청크를 12개씩 병렬로 다운로드합니다
3. **2MB 이상 수신** 시 자동으로 재생이 시작됩니다
4. 백그라운드에서 나머지 청크를 계속 수신하며 **스트리밍 프로그레스 바**가 표시됩니다
5. 모든 청크 수신 완료 시 전체 영상이 로드됩니다

> **참고**: 암호화된 영상은 전체 다운로드 후 복호화하여 재생합니다 (AES-GCM 인증 암호화 특성)

## 보안

- **키 암호화**: API Key는 AES-256-GCM으로 암호화되며 평문으로 저장되지 않습니다
- **읽기 전용**: 잔액 조회 권한만 필요하며 출금이나 거래 권한은 불필요합니다
- **Merkle Tree**: 분산 저장된 청크의 데이터 무결성을 Merkle Root로 검증합니다
- **클라이언트 암호화**: Private 모드 사용 시 서버에서도 내용을 확인할 수 없습니다

## FAQ

**Q: 게시된 콘텐츠를 수정할 수 있나요?**
A: 게시 취소(Unpublish) 후 파일을 수정하고 다시 게시할 수 있습니다.

**Q: 구매한 콘텐츠는 영구적인가요?**
A: 네, 한 번 구매하면 계정에 영구적으로 기록되며 언제든 다시 다운로드할 수 있습니다.

**Q: 암호화된 파일도 판매할 수 있나요?**
A: 기술적으로 가능하지만, 구매자가 파일을 열려면 암호화 비밀번호도 별도로 전달해야 합니다. 일반적으로 판매용 콘텐츠는 암호화 없이 업로드하는 것을 권장합니다.

**Q: 수수료는 어떻게 정산되나요?**
A: 구매 시 Cloud Function이 자동으로 VCN 토큰을 70:30 비율로 분배합니다. 별도의 정산 절차가 필요 없습니다.
